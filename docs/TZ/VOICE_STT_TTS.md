# Voice, STT, TTS

Источник истины: `TZ_Hermes_Mariyam_FINAL_v3_0.md` (v3.2, §0.2, §6)

**Решение заказчика (2026-07-10, ТЗ v3.2): TTS отключён.** Мариям отвечает только текстом (узбекская кириллица); голосовой ввод от Ойижон остаётся. Раздел TTS ниже — на будущее включение.

## Цель

Ойижон должна комфортно пользоваться голосом при вводе. Читать она умеет, но глаза болят, поэтому голосовой ввод — приоритет.

## Поток

Voice в Telegram -> STT (Whisper: сырой текст, допустима латиница/ошибки) -> Hermes/LLM интерпретирует смысл и числа -> при необходимости вызывает tool -> Мариям отвечает текстом кириллицей.

Критерий качества STT — **сквозная точность чисел** (голос -> сохранённая сумма в БД), а не чистота транскрипта.

## Тест реального голоса

До выбора STT собрать 15-20 реальных voice-сообщений Ойижон.

Проверить:

- узбекский и хорезмское произношение;
- числа: `минг`, `миллион`, суммы расходов;
- кириллица/латиница;
- скорость;
- стоимость.

Тест-сет должен включать минимум 10 расходных фраз с числами и минимум 7 медицинских/критических фраз для alerts.

### Протокол теста (Этап 3, пошагово)

1. Получить от Бахриддин ака 15–20 voice-файлов Ойижон (Этап 0). Для каждого записать эталонный текст (узбекская кириллица) и эталонные суммы.
2. Прогнать каждый файл через всех кандидатов STT (список ниже). Фиксировать: распознанный текст, все числа, латиница/кириллица, время ответа, цена за файл.
3. Считать две метрики: (а) WER по тексту — справочно; (б) **точность чисел** — главная: доля фраз, где все суммы распознаны верно. Порог AC: ≥90% на расходных фразах.
4. Отдельно прогнать 7+ медицинских фраз: STT не должен терять ключевые корни (`юрагим оғри`, `нафас ол`...) — иначе keyword-предохранитель не сработает.
5. Результат оформить таблицей «кандидат × метрики», выбрать победителя вместе с заказчиком, зафиксировать выбор в `DECISIONS.md`.
6. Оценку стоимости месяца записать через `log_usage_cost` (прикидка: файлов/день × длина × цена); сверить с бюджетом 10–15 USD/мес.

## STT кандидаты

- ElevenLabs Scribe — рекомендован как базовый кандидат, если тест подтвердит точность.
- Hermes native STT/Whisper.
- OpenAI/Groq Whisper API.
- Local Whisper как fallback.

При неуверенности в сумме Hermes переспросит и не сохранит неверную сумму молча.

## TTS

Голос Мариям: женский, мягкий, спокойный, понятный, не роботизированный. Кандидаты: ElevenLabs, OpenAI TTS, Edge TTS, Hermes-supported TTS. Выбор только после прослушивания на узбекской кириллице.

## LLM и бюджет

Одна основная сильная мультиязычная модель с хорошим узбекским. Не строить сложный per-message router.

Цель бюджета: **10–15 USD/мес**. Если прогноз выше: меньше voice-output, дешёвая модель для простого, кэш новостей/погоды/намаза, учёт через `log_usage_cost` и таблицу `usage_costs`.
